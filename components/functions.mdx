---
title: "Functions"
description: "Dagger functions are the building blocks of your workflows. They can be composed together to create complex pipelines."
---

When you write a Dagger Module, you group your code into **functions**. A function is a piece of code that can be executed by the Dagger Engine, and it can return a value or perform an action (e.g creating and returning a base container, or publishing a container to a registry).
